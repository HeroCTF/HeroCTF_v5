# HeroCTF V5

HeroCTF is an online cybersecurity competition for beginners to advanced players that takes place once a year.

Start : 12/05 at 9pm (UTC+2)<br>
End : 14/05 at 11pm (UTC+2)

Website : https://www.heroctf.fr/<br>
