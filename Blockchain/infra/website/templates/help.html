What to do to get started





MelChain

The MelChain is the HeroCTF blockchain. The coin used is the Melcoin ($MEL).
It runs PoA, which is beneficial for the environment when compared to PoW, but it's also less decentralized.

The node are running at these addresses : http://blockchain.heroctf.fr:8501 & http://blockchain.heroctf.fr:8502 (The fallback link is http://62.171.185.249:[8501-8502]/ ; DNS doesn't like JSON RPC ?)

Important preliminary :

Please create an address for the occasion. Try not to use an address that has funds in it on the MelChain network.
DO NOT COMMUNICATE ANY OF YOUR PRIVATE KEYS TO ANYONE.
DO NOT EVER SEND ANY FUND TO A HEROCHAIN ADDRESS : IT WILL BE YOUR LOSS

You'll never be as rich as Mel, ever, don't bother.

Add to Metamask

To add the Herochain to Metamask, follow these steps :
        - Go into Metamask settings -> networks and click on "Add Network"
        - Enter the following informations :
                Network name     : Herochain
                New RPC URL      : http://blockchain.heroctf.fr:8502 OR http://62.171.185.249:8502/
                Chain ID               : 1337
                Currency Symbol : HERO
                Block explorer : Leave empty (it would make it too easy right ?)

Get money

Go into the faucet section of this website and enter your address. 
The faucet will give you 5 MEL if your balance is less than 1 MEL.

Connect on the web3

You can use the python web3 module (https://web3py.readthedocs.io/en/stable/)
If you prefer JS, there are some good libraries too (https://docs.ethers.io/v5/ , https://web3js.readthedocs.io/en/v1.7.3/)

Remix is also available via Metamask.

Solidity

The smart contracts for these challenges are written in Solidity. You can find more info here : https://docs.soliditylang.org/en/v0.8.14/

For the Blockchain-OSINT challenges, more info is available here : https://ethereum.org/en/developers/docs/apis/json-rpc/ and https://geth.ethereum.org/
